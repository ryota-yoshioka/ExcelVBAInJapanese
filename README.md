# ExcelVBAInJapanese
Excel VBA 日本語化

Excel VBA の日本語化ラッパーです。
- RangeオブジェクトやFile System Object等のクラス名、メソッド名、引数名を日本語したラッパークラスを記載したマクロ有効テンプレートを作成
- 一部の引数つきメソッドを、複数の引数なしメソッドにオーバーロード化

## やる事リスト
- Excel
    - セル範囲（Range）-> GitHub Copilot でソース生成済み（未実装あり）、Unit Test中
    - セル（セル範囲から派生）
    - ワークシート
    - ワークブック
- Scripting
    - ファイルシステムオブジェクト
    - ファイル -> コーディング済み
    - フォルダ
    - ドライブ
    - テキストストリーム -> コーディング済み
- 共通関数
